Project: Traffic Light 4 Arah Arduino

Deskripsi:
Simulasi lampu lalu lintas 4 arah menggunakan Arduino Uno dan GPIO.

Fitur:
- 4 arah (Utara, Timur, Selatan, Barat)
- Sistem looping otomatis
- Tidak ada konflik lampu
